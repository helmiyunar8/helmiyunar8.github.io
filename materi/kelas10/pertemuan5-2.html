FUNGSI OPERATING SYSTEM

Operating System memiliki banyak fungsi penting dalam operasional sebuah komputer. Kita akan bahas satu per satu dengan contoh nyata agar kalian lebih paham.

1. Menjalankan Aplikasi

OS memungkinkan kita untuk membuka berbagai program seperti:

  • Google Chrome
  • Microsoft Word
  • YouTube

👉 Tanpa OS, aplikasi tidak bisa dijalankan sama sekali.

2. Mengatur Hardware

OS bertugas mengatur perangkat keras agar berfungsi:

  • Keyboard agar bisa mengetik
  • Mouse agar bisa klik
  • Layar menampilkan gambar
  • WiFi agar bisa internetan

3. Mengatur File & Data

OS membantu kalian dalam pengelolaan data:

  • Menyimpan file tugas sekolah
  • Membuka file foto/gambar
  • Menghapus file video

4. Mengatur Jaringan

Penting untuk materi jaringan komputer:

  • Menghubungkan ke WiFi
  • Mengatur alamat IP Address
  • Mengakses jaringan internet

5. Keamanan Sistem

OS bertindak sebagai pelindung:

  • Melindungi data pengguna
  • Menjaga file penting
  • Password login & hak akses

📌 Kesimpulan Halaman 2

Operating System adalah pengatur semua aktivitas komputer, mulai dari aplikasi, perangkat keras, hingga jaringan yang kalian gunakan.
