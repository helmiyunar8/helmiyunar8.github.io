🖥️ JENIS-JENIS OPERATING SYSTEM

Berikut adalah OS yang sering digunakan dalam kehidupan sehari-hari.

💻 1. Windows

OS yang paling umum digunakan di sekolah dan kantor.

Ciri-ciri: Tampilan GUI mudah, banyak aplikasi pendukung.

💻 2. Chrome OS

OS yang digunakan pada Chromebook di SMKN 7 LUWU.

Ciri-ciri: Ringan, cepat, dan fokus pada layanan Cloud/Internet.

📱 3. Android

OS berbasis Linux untuk perangkat mobile/smartphone.

Ciri-ciri: Open source, jutaan aplikasi di Play Store.

🍎 4. macOS

Sistem operasi eksklusif milik perangkat Apple.

Ciri-ciri: Sangat stabil, tampilan elegan, untuk profesional.

🐧 5. Linux

OS favorit para teknisi jaringan dan programmer.

Ciri-ciri: Gratis, aman dari virus, bisa dimodifikasi (Distro).

📌 Kesimpulan Halaman 3

Meskipun berbeda tampilan, semua OS berfungsi sebagai jembatan antara user dan hardware.
