🌍 PERTEMUAN 6

🏠 JENIS-JENIS IP ADDRESS

Tidak semua IP Address itu sama. Berdasarkan jangkauan dan cara pemberiannya, IP Address dibagi menjadi beberapa kelompok utama yang wajib dipahami oleh teknisi jaringan.

1. IP Public

IP yang digunakan untuk berkomunikasi di jaringan internet global. Ibarat nomor telepon internasional yang bisa dihubungi dari mana saja.

Contoh: 8.8.8.8 (Google)

  • Bisa diakses dari seluruh dunia.
  • Diberikan oleh ISP (Indihome, Telkomsel, dsb).

2. IP Private

Digunakan khusus untuk jaringan lokal (LAN) seperti di dalam Lab sekolah kita. Tidak bisa diakses langsung dari internet luar.

Range IP Private:

• 10.x.x.x
• 172.16.x.x - 172.31.x.x
• 192.168.x.x

Analogi di Sekolah:

Chromebook kalian di lab menggunakan IP Private (lokal), sedangkan koneksi internet utama sekolah kita memiliki IP Public untuk keluar ke internet.

🔧 Cara Pemberian IP

Selain jenisnya, cara perangkat mendapatkan IP juga berbeda-beda:

3. DHCP (Otomatis)

IP diberikan secara otomatis oleh Router. Kalian tidak perlu setting apa-apa, cukup hubungkan WiFi dan langsung dapat IP.

Contoh: Saat kalian connect WiFi HP atau Chromebook sekolah.

4. Static (Manual)

IP diatur secara manual oleh admin/teknisi. Alamatnya tidak akan berubah-ubah (tetap).

Digunakan untuk:

  • Server Web/Sekolah
  • Printer Jaringan
  • Setting Access Point Utama

📌 Kesimpulan Halaman 3

Secara garis besar, IP dibagi menjadi dua sisi: Public & Private (berdasarkan jangkauan) serta DHCP & Static (berdasarkan cara pengaturannya).
