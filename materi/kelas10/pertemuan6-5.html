🌍 PERTEMUAN 6

🧪 PRAKTIK IP ADDRESS (CHROMEBOOK)

Tujuan Praktik:

Agar siswa mampu melakukan konfigurasi IP secara manual dan memahami alur koneksi jaringan pada Chromebook.

⚙️ Langkah-Langkah Konfigurasi

01. Klik pada ikon WiFi/Jam di pojok kanan bawah layar Chromebook kalian.
02. Pilih ikon tanda panah di samping nama WiFi yang terhubung, lalu klik "Network" atau Pengaturan Jaringan.
03. Scroll ke bawah hingga menemukan bagian Network. Ubah opsi "Configure IP address automatically" (DHCP) menjadi "Static IP".

📌 Masukkan Data Berikut:

Silakan isi kolom pada Chromebook kalian sesuai dengan tabel di bawah ini:

IP Address 192.168.1.10
Subnet Mask 255.255.255.0
Gateway 192.168.1.1
DNS Server 8.8.8.8

🌐 Uji Koneksi

Setelah selesai mengisi, cobalah buka browser Google Chrome dan akses situs google.com. Jika terbuka, berarti konfigurasi kalian berhasil!

Troubleshooting (Jika Gagal):

  • Cek IP: Pastikan tidak ada salah ketik angka atau titik.
  • Cek Gateway: Pastikan alamat gateway mengarah ke router yang benar.
  • Cek DNS: Pastikan DNS Server terisi (8.8.8.8).

📌 Kesimpulan Halaman 5

Melalui praktik ini, siswa dapat melihat langsung bagaimana teori IP Address yang dipelajari sebelumnya diterapkan secara nyata pada perangkat kerja.
