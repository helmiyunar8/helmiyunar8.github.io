Pembahasan Tryout TKJ

Halaman 2 (Soal 11–20)
11. Konektor kabel LAN

Jawaban: RJ45

RJ45 adalah konektor yang digunakan pada kabel UTP untuk jaringan LAN.
12. Server pembatas akses internet

Jawaban: Proxy Server

Proxy server berfungsi mengontrol dan membatasi akses internet pengguna.
13. Perangkat pada gambar

Jawaban: Switch

Switch menghubungkan perangkat dalam jaringan LAN dan mengatur lalu lintas data.
14. Perangkat jaringan pada gambar

Jawaban: Modem

Modem digunakan untuk mengubah sinyal digital menjadi analog agar dapat dikirim melalui jaringan ISP.
15. Penguat sinyal jaringan

Jawaban: Repeater

Repeater digunakan untuk memperkuat sinyal jaringan agar dapat menjangkau jarak lebih jauh.
16. Internet sebagai media informasi

Jawaban: Media informasi

Internet mempermudah penyebaran informasi secara global.
17. Dampak negatif internet

Jawaban: Konten negatif

Internet dapat memberikan dampak negatif jika tidak digunakan secara bijak.
18. Jaringan menggunakan kabel disebut

Jawaban: Wireline

Wireline adalah jaringan yang menggunakan media kabel.
19. Perawatan hardware VoIP

Jawaban: A,C,D benar

Perawatan meliputi pengecekan perangkat, membersihkan debu, dan memeriksa periferal.
20. Layanan penyimpanan website

Jawaban: Web Hosting

Web hosting adalah layanan penyimpanan file website agar dapat diakses melalui internet.
