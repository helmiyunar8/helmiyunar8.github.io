🏆 Pembahasan Tryout TKJ

11. Konektor kabel LAN?
Jawaban: RJ45
RJ45 adalah konektor standar yang digunakan pada kabel UTP (Unshielded Twisted Pair) untuk membangun jaringan komputer lokal (LAN).
12. Server pembatas akses internet?
Jawaban: Proxy Server
Proxy server berfungsi sebagai perantara antara pengguna dan internet, sekaligus mengontrol, memfilter, dan membatasi akses internet bagi pengguna di dalam jaringan.
13. Perangkat pada gambar? (Switch)
Jawaban: Switch
Switch menghubungkan banyak perangkat dalam jaringan LAN dan bekerja lebih cerdas dibandingkan Hub dengan mengatur lalu lintas data berdasarkan alamat MAC.
14. Perangkat jaringan pada gambar? (Modem)
Jawaban: Modem
Modem (Modulator-Demodulator) digunakan untuk mengubah sinyal digital dari komputer menjadi analog agar dapat dikirim melalui jalur penyedia layanan internet (ISP).
15. Penguat sinyal jaringan?
Jawaban: Repeater
Repeater adalah alat yang bertugas menangkap sinyal lemah dan memancarkannya kembali dengan kekuatan penuh agar jangkauan jaringan bisa lebih luas.
16. Internet sebagai media informasi?
Jawaban: Media Informasi
Internet mempermudah akses dan penyebaran informasi secara instan dan global, menjadikannya salah satu media informasi paling utama saat ini.
17. Dampak negatif internet?
Jawaban: Konten Negatif
Kemudahan akses informasi di internet juga membawa risiko paparan konten negatif, berita bohong (hoax), hingga kejahatan siber jika tidak digunakan secara bijak.
18. Jaringan menggunakan media kabel disebut?
Jawaban: Wireline
Wireline adalah sistem komunikasi atau jaringan yang menggunakan media fisik berupa kabel (tembaga atau fiber) untuk mentransmisikan data.
19. Perawatan hardware VoIP?
Jawaban: A, C, dan D Benar
Perawatan VoIP meliputi: (A) Pengecekan perangkat secara berkala, (C) Membersihkan debu agar tidak panas, dan (D) Memeriksa kondisi fisik periferal pendukung.
20. Layanan penyimpanan website?
Jawaban: Web Hosting
Web hosting adalah layanan penyewaan ruang di server agar file-file website dapat disimpan dan diakses oleh orang lain melalui jaringan internet 24 jam non-stop.
