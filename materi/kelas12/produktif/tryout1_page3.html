🏆 Pembahasan Tryout TKJ

21. Data hosting website?
Jawaban: Database HTML
Website di internet disimpan dalam bentuk file HTML (untuk struktur) dan Database (untuk data dinamis) pada media penyimpanan server hosting.
22. SLD blog bilacilasila.blogspot.com?
Jawaban: bilacilasila
SLD (Second Level Domain) adalah nama unik yang Anda buat sendiri. Dalam contoh ini, 'bilacilasila' adalah SLD-nya.
23. TLD pada blogspot.com?
Jawaban: .com
TLD (Top Level Domain) adalah akhiran dari nama domain. Contoh umum adalah .com, .net, .org, atau .id.
24. Struktur domain blogspot?
Jawaban: D
Struktur domain yang lengkap biasanya terdiri dari Subdomain (contoh: www), Domain Utama (SLD), dan TLD.
25. Pengertian domain?
Jawaban: Domain
Domain adalah nama unik yang diberikan untuk mengidentifikasi alamat IP server komputer di jaringan internet.
26. Hosting satu server banyak domain?
Jawaban: Shared Hosting
Shared Hosting adalah layanan hosting di mana satu server digunakan bersama-sama oleh banyak pengguna untuk menekan biaya.
27. Server dititip di data center?
Jawaban: Colocation
Colocation Hosting adalah layanan di mana pelanggan menitipkan server miliknya sendiri di gedung Data Center resmi.
28. Fungsi Plesk?
Jawaban: A, B, D Benar
Plesk adalah control panel hosting yang berfungsi untuk mengelola hosting, bisa berjalan di Linux/Windows, dan mengotomatisasi tugas server.
29. Domain .gov digunakan untuk?
Jawaban: Pemerintah
Domain .gov (Government) dikhususkan untuk situs-situs resmi milik lembaga pemerintah atau instansi negara.
30. Domain .edu digunakan untuk?
Jawaban: Pendidikan
Domain .edu (Education) digunakan secara eksklusif oleh institusi pendidikan seperti universitas atau lembaga riset.
