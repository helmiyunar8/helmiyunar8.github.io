Pembahasan Tryout TKJ

Halaman 4 (Soal 31–40)
31. VLAN

Jawaban: Virtual LAN

VLAN adalah teknologi jaringan yang memungkinkan pembagian jaringan secara logis dalam satu switch.
32. VLAN untuk data user

Jawaban: Data VLAN

Data VLAN digunakan untuk lalu lintas data normal dari pengguna jaringan.
33. VLAN manajemen switch

Jawaban: Management VLAN

Management VLAN digunakan untuk konfigurasi dan pengelolaan perangkat jaringan.
34. Routing statis

Jawaban: Manual

Routing statis adalah metode routing yang dikonfigurasi secara manual oleh administrator jaringan.
35. Dynamic routing

Jawaban: Router membuat tabel routing otomatis

Dynamic routing menggunakan protokol seperti RIP atau OSPF untuk memperbarui tabel routing.
36. Gateway adalah

Jawaban: Gerbang jaringan

Gateway adalah perangkat yang menghubungkan dua jaringan dengan protokol berbeda.
37. Firewall adalah

Jawaban: Keamanan jaringan

Firewall berfungsi melindungi jaringan dengan memfilter lalu lintas data masuk dan keluar.
38. QoS

Jawaban: Quality of Service

QoS digunakan untuk mengatur prioritas lalu lintas jaringan agar layanan penting tetap stabil.
39. Load Balancing

Jawaban: Distribusi trafik

Load balancing membagi beban jaringan ke beberapa server agar kinerja lebih stabil.
40. Fungsi Firewall

Jawaban: Filter trafik

Firewall menyaring paket data yang masuk dan keluar dari jaringan untuk menjaga keamanan.
