🏆 Pembahasan Tryout TKJ

1. Apa yang dimaksud routing?
Jawaban: Perangkat menghubungkan dua jaringan
Routing adalah proses menentukan jalur terbaik untuk mengirim paket data dari satu jaringan ke jaringan lain. Perangkat yang melakukan proses ini adalah router.
2. Perangkat yang menghubungkan komputer dalam jaringan?
Jawaban: Switch
Switch menghubungkan beberapa komputer dalam jaringan LAN dan mengirimkan data langsung ke perangkat tujuan menggunakan alamat MAC.
3. Perangkat PCB untuk koneksi jaringan?
Jawaban: NIC
NIC (Network Interface Card) adalah kartu jaringan yang dipasang pada komputer agar dapat terhubung ke jaringan.
4. Perangkat yang bukan hardware jaringan?
Jawaban: Gateway
Gateway merupakan konsep penghubung antar jaringan yang menggunakan protokol berbeda. Biasanya berupa software atau konfigurasi pada router.
5. Perangkat yang meneruskan paket data?
Jawaban: Router
Router bertugas meneruskan paket data dari satu jaringan ke jaringan lain menggunakan tabel routing sebagai panduan.
6. Perangkat menyalurkan data ke tujuan saja?
Jawaban: Switch
Berbeda dengan Hub yang membroadcast data, Switch mengirimkan data hanya ke perangkat tujuan sehingga lalu lintas jaringan lebih efisien.
7. Cara kerja server melayani client disebut?
Jawaban: Client Server
Model client-server adalah arsitektur jaringan dimana server menyediakan layanan/sumber daya dan client meminta layanan tersebut.
8. Perangkat jaringan pada gambar?
Jawaban: Router
Router digunakan untuk menghubungkan dua jaringan atau lebih dan menentukan jalur pengiriman data tercepat (routing).
9. Jumlah layer OSI?
Jawaban: 7 Layer
Model OSI terdiri dari 7 lapisan: Physical, Data Link, Network, Transport, Session, Presentation, Application.
10. Urutan warna kabel UTP (Standar T568B)?
Jawaban: T568B
Putih Orange – Orange – Putih Hijau – Biru – Putih Biru – Hijau – Putih Coklat – Coklat.
