Pembahasan Tryout TKJ

Halaman 1 (Soal 1 - 10)
1. Apa yang dimaksud routing?

Jawaban Benar: Perangkat menghubungkan dua jaringan

Pembahasan:
Routing adalah proses menentukan jalur terbaik untuk mengirim paket data antar jaringan. Perangkat yang melakukan proses ini adalah router.
2. Perangkat yang menghubungkan komputer dalam jaringan

Jawaban Benar: Switch

Pembahasan:
Switch digunakan dalam jaringan LAN untuk menghubungkan banyak komputer dan meneruskan data berdasarkan alamat MAC.
3. Perangkat PCB untuk koneksi jaringan

Jawaban Benar: NIC

Pembahasan:
NIC (Network Interface Card) adalah kartu jaringan yang dipasang pada motherboard komputer agar dapat terhubung ke jaringan.
4. Perangkat yang bukan hardware jaringan

Jawaban Benar: Gateway

Pembahasan:
Gateway adalah konsep penghubung jaringan yang menggunakan protokol berbeda, tidak selalu berupa perangkat fisik.
5. Perangkat yang meneruskan paket data

Jawaban Benar: Router

Pembahasan:
Router berfungsi meneruskan paket data dari satu jaringan ke jaringan lain menggunakan tabel routing.
6. Perangkat menyalurkan data ke tujuan saja

Jawaban Benar: Switch

Pembahasan:
Switch hanya mengirim data ke port tujuan sesuai alamat MAC sehingga lebih efisien dibanding hub.
7. Cara kerja server melayani client disebut



Jawaban Benar: Client Server

Pembahasan:
Model client-server adalah sistem dimana server menyediakan layanan dan client meminta layanan melalui jaringan.
8. Perangkat jaringan pada gambar



Jawaban Benar: Router

Pembahasan:
Router menghubungkan dua atau lebih jaringan dan menentukan jalur pengiriman paket data.
9. Jumlah layer OSI

Jawaban Benar: 7 Layer

Pembahasan:
Model OSI memiliki 7 lapisan yaitu: Physical, Data Link, Network, Transport, Session, Presentation, dan Application.
10. Urutan warna kabel UTP

Jawaban Benar: Standar T568B

Pembahasan:
Putih Orange – Orange – Putih Hijau – Biru – Putih Biru – Hijau – Putih Coklat – Coklat.
